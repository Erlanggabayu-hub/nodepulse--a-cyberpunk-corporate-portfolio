# Erlangga Bayu - Web3 Ambassador Portfolio

A futuristic, professional personal portfolio website for Erlangga Bayu, a Web3 Ambassador specializing in DePIN and Blockchain Infrastructure.

## Framework

Vite 6.x + React 19.x

## Design

**Theme:** Cyberpunk Corporate  
**Aesthetic:** Dark mode with neon green (#00ff88) and electric blue (#00d9ff) accents  
**Typography:** Inter (sans-serif) and Space Mono (monospace)

The portfolio features:
- Hero section with compelling headline
- Featured NodeOps Network showcase with connected nodes visualization
- Ambassador timeline with key roles and milestones
- Core competencies displayed as interactive tech badges
- About section emphasizing Web3 expertise and community focus
- Professional footer with social links and contact information

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access with secure subdomain handling via `.nodeops.app` allowedHosts configuration.
